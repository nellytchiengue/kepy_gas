## Auto-Depannage

Avant de nous contacter, verifiez ces points:

### Probleme: Menu "Factures" invisible
**Solution:**
1. Rafraichissez la page (F5)
2. Attendez 10 secondes
3. Si toujours absent, fermez et rouvrez le Sheet

### Probleme: Autorisation Google echoue
**Solution:**
1. Verifiez que vous utilisez le meme compte Google
2. Essayez en navigation privee
3. Desactivez temporairement les extensions

### Probleme: Template non trouve
**Solution:**
1. Relancez l'assistant d'installation
2. Verifiez les permissions Drive
3. Le template doit etre dans le meme dossier que le Sheet

### Probleme: PDF ne se genere pas
**Solution:**
1. Verifiez que l'ID de facture est unique
2. Verifiez que tous les champs obligatoires sont remplis
3. Consultez le message d'erreur dans la popup

### Probleme: Email non envoye
**Solution:**
1. Verifiez l'adresse email client
2. Verifiez les permissions Gmail
3. Consultez le dossier "Sent" de Gmail
