# InvoiceFlash - Installation

Bienvenue dans InvoiceFlash ! Ce dossier contient tout le necessaire pour installer et configurer votre systeme de facturation.

| Fichier | Description |
|---------|-------------|

## Installation Rapide (5 minutes)

### Etape 1: Upload du Google Sheet
1. Connectez-vous a Google Drive (drive.google.com)
2. Cliquez sur "Nouveau" > "Importer un fichier"
3. Selectionnez `InvoiceFlash_Sheet.xlsx`
4. Attendez la conversion automatique en Google Sheets

### Etape 2: Lancement de l'assistant
1. Ouvrez le fichier converti dans Google Sheets
2. Attendez 5-10 secondes que le menu apparaisse
3. Cliquez sur "Factures" > "Installation"
4. Suivez les 6 etapes de l'assistant

### Etape 3: Configuration
1. Selectionnez votre pays (France, Cameroun, USA)
2. Renseignez vos informations d'entreprise
3. Ajoutez vos coordonnees bancaires
4. (Optionnel) Uploadez votre logo

### Etape 4: Premiere facture
1. Cliquez sur "Factures" > "Nouvelle facture"
2. Remplissez les informations client et service
3. Cliquez sur "Enregistrer"
4. Cliquez sur "Generer" pour creer le PDF

## Problemes courants

**Le menu "Factures" n'apparait pas**
→ Rafraichissez la page (F5 ou Cmd+R)

**Erreur d'autorisation Google**
→ Utilisez le meme compte Google pour Sheet et Drive

**Template non trouve**
→ L'assistant le cree automatiquement, relancez l'installation
