## 5. FAQ PRE-ACHAT / POST-ACHAT

### FAQ Installation

**Q: Je ne vois pas le menu "Factures" apres ouverture du Sheet**
R: Rafraichissez la page (F5 ou Cmd+R). Le menu peut prendre quelques secondes a apparaitre.

**Q: L'autorisation Google echoue**
R: Assurez-vous d'utiliser le meme compte Google pour le Sheet et le Drive. Si le probleme persiste, essayez en navigation privee.

**Q: Ou trouver mon SIRET / NIU / EIN ?**
R:
- France (SIRET): site sirene.fr ou vos documents d'immatriculation
- Cameroun (NIU): Centre des impots ou carte contribuable
- USA (EIN): Lettre IRS ou site irs.gov

**Q: Puis-je utiliser InvoiceFlash sur plusieurs comptes Google ?**
R: Chaque installation est liee a un compte Google. Pour utiliser sur plusieurs comptes, dupliquez le Sheet et reinstallez sur chaque compte.

### FAQ Utilisation

**Q: Combien de factures puis-je generer ?**
R: Illimite. Pas de restriction sur le nombre de factures.

**Q: Puis-je personnaliser le design de la facture ?**
R: Oui. Le template est un Google Docs standard que vous pouvez modifier librement (couleurs, polices, logo, mise en page).

**Q: Les donnees sont-elles securisees ?**
R: Oui. Tout reste sur votre Google Drive personnel. Aucune donnee n'est envoyee a des serveurs externes.

**Q: Puis-je facturer dans plusieurs devises ?**
R: Oui. Changez simplement le pays dans les parametres pour adapter la devise (EUR, USD, XAF, etc.).

### FAQ Technique

**Q: Quel navigateur utiliser ?**
R: Chrome est recommande. Firefox et Edge fonctionnent aussi. Safari peut avoir des limitations.

**Q: InvoiceFlash fonctionne-t-il sur mobile ?**
R: Partiellement. L'interface est optimisee pour desktop. Vous pouvez consulter le Sheet sur mobile mais la generation de factures est plus confortable sur ordinateur.

**Q: Y a-t-il une limite de stockage ?**
R: Non de notre cote. Votre limite est celle de votre Google Drive (15 Go gratuits par defaut).
